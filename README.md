# Urban Route Optimization using Hybrid Genetic Algorithm and Ant Colony Optimization

## Overview

This project explores different metaheuristic approaches for solving urban route optimization problems.

Three approaches were implemented and compared:

- Genetic Algorithm (GA)
- Ant Colony Optimization (ACO)
- Hybrid GA + ACO

The objective is to minimize route cost while maintaining computational efficiency.

## Technologies

- Python
- NumPy
- Folium
- OpenStreetMap

## Results

The hybrid approach achieved the best trade-off between solution quality and execution time.
